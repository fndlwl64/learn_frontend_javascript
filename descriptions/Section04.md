## Section04
### DOM API
- 목적: 자바스크립트로 HTML 문서를 조작
    - HTML 요소를 객체(DOM 객체)로 변환
    - Node를 상속받는 객체
- API
    - 노드 생성
        - querySelector: 요소 접근 
        - createElement: 요소 생성
    - 요소 내 속성 및 내용 접근
        - classList: 요소 내에서 class 속성에 접근
        - textContent: 요소 내에 문자열에 접근
        - appendChild: DOM 객체에 자식 노드 추가 
        - style: 스타일에 접근
    - 이벤트 처리
        - addEventListner: 요소에 이벤트 추가
